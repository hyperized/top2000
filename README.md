# Top 2000

Slack bot that shows you what Top2000 song is currently playing.

### Fetch

```
go get github.com/hyperized/top2000
```

### Build

```
go build github.com/hyperized/top2000
```

### Configure

```
cat <<EOF > config.yml
api:
  token: "my-slack-bot-token-here"
EOF
```

### Run

```
./top2000
```
